# ainativemeo.com

LLMO Framework のローカルビジネス向けリファレンス実装サイト。Astro v6 + Tailwind v4、GitHub Pages デプロイ。

## 開発

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # dist/ にビルド
npm run preview  # ビルド結果のプレビュー
```

## デプロイ

`main` ブランチへの push で GitHub Actions が自動デプロイ。Custom domain は `public/CNAME` に記載。
